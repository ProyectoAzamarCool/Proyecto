# Proyecto
Aloha, compañeros, vamos a comenzar revisando si pueden interactuar con el repositorio: Van a copiar este repo y van a crear su rama, luego la van a intentar subir y veremos si es posible o no. 
Cualquier observación envienla por wa.

Por el momento, hay 2 ramas: main y develop. 
Main es la rama que tendra el producto "final" en todo momento, es decir, que todo sea estable y funcione correctamente, por ende los cambios no se realizaran en ella. ESTA RAMA NO SE TOCAAAAA! 
develop: es la rama donde se probaran todas las adiciones hechas en un determinado tiempo, cada rama que se cree debe tener a develop como rama de origen, por el contrario nos podría cargar la verga.
